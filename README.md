# Credit-Scoring-Model
A Machine Learning project that predicts an individual's creditworthiness using financial data. The model is trained with classification algorithms and evaluated using Accuracy, Precision, Recall, F1-Score, and ROC-AUC. 


## Overview
This project predicts an individual's creditworthiness using Machine Learning classification algorithms. It helps determine whether a customer is likely to be creditworthy based on financial information.

## Features
- Data preprocessing
- Feature engineering
- Model training and evaluation
- Performance metrics:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
  - ROC-AUC

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains customer financial information such as income, debt, payment history, and other related features.
## Results
The model is evaluated using standard classification metrics to measure prediction performance.
